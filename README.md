# PhishGuard — Phishing Awareness Platform

An interactive cybersecurity awareness platform that demonstrates how phishing attacks work and teaches users how to protect themselves.

Built as part of my cybersecurity portfolio during Ethical Hacking training at EyeQ Dot Net Pvt. Ltd.

---

## Live Demo

> Clone the repo, open `index.html` in your browser and explore all pages.

---

## Pages

| Page | Description |
|---|---|
| Live Demo | Experience a simulated phishing login page firsthand |
| Busted Page | See exactly what credentials were "captured" and why it worked |
| Techniques | Learn 6 real phishing techniques used by attackers today |
| URL Analysis | Interactive tool to detect suspicious URLs with real-time analysis |
| Email Analysis | Annotated phishing email with clickable red flags and safety checklist |

---

## Features

- Simulated phishing login page with credential capture demo
- Real-time URL suspicion checker detecting typosquatting, subdomain tricks, suspicious keywords, URL shorteners
- Interactive phishing email analyser with clickable annotations
- Phishing checklist tool with instant risk assessment
- Covers 6 phishing types — Spear Phishing, Smishing, Vishing, Clone Phishing, Whaling, Pharming
- Clean responsive UI with navbar navigation
- Educational disclaimer on every page
